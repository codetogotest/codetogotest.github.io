學習用 Hugo 製作個人網站

By Aki 2024-05-12 Learning #Tools #Hugo

在學習 Hugo 過程中，有遇到一些小問題，也學習找出問題後，一個一個的排除問題，最後也有執行成功

感謝古古老師教學，有興趣的人可以至老師網站參考https://kucw.io/

將學習過程中簡略的整理，會使用到的工具或一些設定相關的連結

Hugo https://gohugo.io/

下載 Hugo 工具 (Prebuilt binaries 下點擊 latest release) https://github.com/gohugoio/hugo/releases/tag/v0.125.7

下載後的檔案進行解壓縮，將解壓縮後裡面的 exe 執行檔案，放置要產出網站檔案夾裡面就好

Hugo themes https://themes.gohugo.io/

有遇到選好 themes 模版是無法使用，只好先將課程所選的模版做個人部落格使用，先理解整個架樣後，套用成功後再套用其他的模版，比較能釐清問題可能會發生在哪邊，才能快速解決問題及排除問題

課程中是選用這個 Hugo themes 模版, 使用 Liva Hugo https://github.com/gethugothemes/liva-hugo

Configure rendering of markup to HTML https://gohugo.io/getting-started/configuration-markup/#highlight

markup Chroma Style Gallery https://xyproto.github.io/splash/docs/all.html

https://gohugo.io/methods/time/format/

在修改程式內容時，有遇到修改日期部份，記得老師有提到，但自己在邊學邊改時沒注意到，結果狂出錯，因轉出來格式永遠都不是正確的，光這個地方讓人快崩潰，但覺得很疑惑為何會錯，再次聽課程內容才發現到，真的有差別，老師說要選用 Format string 下裡面日期格式內容 copy 自己的程理內，但我卻使用旁邊的 Result，看起來是一樣，但上去就完全不一樣，也是經驗學習

在 Hugo 模版上加評論，是使用第三方業者，課程教學是使用 disqus，是需要至 disqus 做註冊後才能使用

disqus https://disqus.com/

utterances https://utteranc.es/

基本上就完成一大半的工程，剩下是自己在使用 VS Code 工具在編輯 HTML 檔案後，存檔過程中就有告知 HTML 格式有問題，因自己也有安裝其他相關的工具，此問題就自行找設定配置問題再重新設定將問題解決，學習的過程中真的會遇到各式各樣的狀況，但看到老師都會一一回覆同學，真的是很貼心與用心的老師

第一次接觸 Hugo 與 Markdown 是需要一些時間，遇到各式狀況，也是學習如何解決問題，但很開心是美好的，也順利完成，剩下的文章會陸續再補上內容
